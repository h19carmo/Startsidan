Sebastian Mortensen Östman

Hej mitt namn är Sebastian och jag kommer från en stad som heter Kumla, flyttade därefter till Örebro. I Örebro studerade jag på ett gymnasium där jag gick ett program som hette game design. Det handlade mycket om information design och layout etc. På min fritid sitter jag oftast framför datorn och spelar eller pratar med andra via sociala medier. De program som jag använder mig utav då är Discord eller Twitch. De två går rätt hand i hand eftersom det är stora communities från Twitch som använder Discord som kommunikation etc. Jag kollar även mycket på anime som jag fastnade för när jag var väldigt ung, den största influensen till det var pokemon och beyblade.
